# Robocorp USD → COP

Este robot obtiene el valor actual del dólar frente al peso colombiano usando una búsqueda en Google.

## Pasos

1. Clona este repositorio
2. Abre VS Code con la extensión de Robocorp
3. Ejecuta `Run Task → get_usd_cop`
4. Verás el valor impreso o en los outputs del robot

## Outputs

- `exchange_rate`: texto con el valor del dólar frente al peso colombiano
